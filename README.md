# Unified-Health
Proyecto del ramo "Desarrollo de aplicaciones moviles en android", especificamente la parte del front end del proyecto, para revisar el backend presione el siguiente [link](https://github.com/Haidex3/Unified-Health-Backend).

## Descripción
proyecto pensado para gestionar las multiples informaciones medicas de los hospitales y sus pacientes para facilitar el acceso de la informacion.

## Instalación

- Descargar este repositorio
- Descargar repositorio [backend](https://github.com/Haidex3/Unified-Health-Backend)
- Descomprimir ambos repositorios en sus carpetas especificas
- En el repositorio de backend, instalas los componentes faltantes con el comando "laravel install"
- En el repositorio de backend, migrar la base de datos con el comando "compose migrate" 
- En el repositorio de backend, arracar el servidor con el comando "php serve"
- En el repositorio de frontend, abrir el proyecto desde android studio
- En el repositorio de frontend, actualizar gradle 
- En el repositorio de frontend, verificar que la carpeta c./windows/TEMP/ es editable (si accede a esta direccion puede verificarlo)
- En el repositorio de frontend, ejecutar el proyecto, ya sea via emulador o por cable hacia un dispositivo.
